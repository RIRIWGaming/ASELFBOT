### CARA INSTALL DI TERMUX  <img src="https://github.com/TheDudeThatCode/TheDudeThatCode/blob/master/Assets/hmm.gif" width="29px">
```bash
> termux-setup-storage [ Lalu Izinkan ]
> apt-get update -y
> apt-get upgrade -y
> pkg update -y
> pkg upgrade -y
> pkg install git -y
> pkg install bash -y
> git clone https://github.com/RIRIWGaming/SELFBOT
> cd SELFBOT
> bash install.sh
> npm i
> rm -rf session.json
> npm start
```
